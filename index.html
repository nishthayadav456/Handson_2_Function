Q4.Print output:

var x = 21;
var girl = function () {
console.log(x);
var x = 20;
};
girl ();

→The reason for output being 'undefined' instead of '21' is due to variable hoisting .The variable 'x' is declared inside the 'girl' function .wherever we declare a variable without assigning any value to it, javascript implicitly assigns its value as 'undefined'.
